# Logistic Regression - Classification Model

This project demonstrates Logistic Regression for binary classification using Python.

##  Description
We use logistic regression to predict the target.
The model is built using *scikit-learn* and evaluated using metrics like accuracy, precision, recall, and ROC-AUC.

## Steps
1. Import necessary libraries
2. Load dataset
3. Data preprocessing
4. Train-test split
5. Model training
6. Model evaluation

## Requirements
- Python 3.8+
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
